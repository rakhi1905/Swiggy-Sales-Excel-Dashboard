# Swiggy Sales Analysis Excel Dashboard

## Project Overview
This project analyzes Swiggy food delivery data using Microsoft Excel to generate business insights. An interactive dashboard was created to visualize sales trends, customer ratings, order distribution, and top-performing cities.

The goal of this project is to demonstrate data analysis and visualization skills using Excel tools such as Pivot Tables, Pivot Charts, slicers, and map visualizations.

## Key Insights
• Total Sales: ₹53M+  
• Average Rating: 4.34  
• Average Order Value: ₹268.51  
• Total Orders: 197K+  
• Top Cities by Sales: Bangalore, Lucknow, Hyderabad, Mumbai, New Delhi  

## Tools Used
• Microsoft Excel  
• Pivot Tables  
• Pivot Charts  
• Data Cleaning  
• Data Visualization  

## Dashboard Features
• Monthly Sales Trend  
• Weekly Sales Trend  
• Daily Sales Trend  
• Sales by Food Type  
• Sales by State (Map Visualization)  
• Top Cities by Sales
